## 2. 현재 직면한 심각한 기술 이슈

| 이슈 | 상태 | 원인 및 현상 |
|---|---|---|
| **드라이브 업로드 403** | ❌ 미해결 | 서비스 계정의 저장 용량이 0이며, 사용자 폴더로의 권한 위임이 정상적으로 이루어지지 않음. `utils/drive.py` 코드는 구현 완료. |
| **Gemini OCR 동작 미검증** | ⚠️ 코드완료 | `utils/ocr.py` 재작성 완료 (`gemini-2.5-flash`, JSON 파싱, 키 체크). 실제 영수증 이미지로 동작 검증 필요. |
| **탭 전환 동기화** | ✅ 해결 | 새 탭 열기 방식으로 우회. `window.close()` 복귀 버튼 추가. |
| **AttributeError** | ✅ 해결 | 컬럼 존재 여부 확인 후 합계 계산. 크래시 없음. |

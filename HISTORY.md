| **Gemini OCR 동작 미검증** | ⚠️ 코드완료 | `utils/ocr.py` 재작성 완료 (`gemini-2.5-flash`, JSON 파싱, 키 체크). 실제 영수증 이미지로 동작 검증 필요. |

## 5. 파일 목록 및 역할

- `app.py`: 메인 웹 애플리케이션 진입점. URL 파라미터(`?tab=expenses`) 분기 처리.
- `tabs/prep.py`: 여행 준비 탭 (예산, 체크리스트, 예상일정).
- `tabs/trip.py`: 여행 현지 탭 (지도, 일정 목록, 카테고리 버튼).
- `tabs/expenses.py`: 경비 관리 탭 (직접입력, OCR, 최근 5그룹 뷰어).
- `tabs/history.py`: 여행 경비내역 탭 (일차 필터, 전체 가계부 뷰어).
- `tabs/ai.py`: AI 여행 비서 탭 (thinklog 뷰어, 외부 AI 링크).
- `utils/sheets.py`: 구글 시트 연결·읽기·쓰기. `VND_TO_KRW` 상수. `queue_update` / `flush_queue` / `pending_count` 오프라인 큐.
- `utils/auth.py`: 로그인 로직.
- `utils/ocr.py`: Gemini API 기반 영수증 OCR.
- `utils/drive.py`: 구글 드라이브 파일 업로드 (서비스 계정 방식).
- `scripts/fill_coords.py`: 좌표·장소명 자동 추출 스크립트.
- `scripts/fix_secrets.py`: 구글 서비스 계정 JSON 키 → secrets.toml 자동 변환 도구.
- `start.sh`: 맥 로컬 실행 스크립트 (venv 자동 생성 포함).
- `fill.sh`: 좌표·장소명 자동 채우기 단축 실행 스크립트.
- `.streamlit/secrets.toml`: 구글 시트 서비스 계정 키, 독스 ID, 비밀번호 보관소 (비공개, 깃 제외).

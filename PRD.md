# 제품 요구사항 정의서 (PRD): 베트남 스마트 여행 플래너

## 1. 개요
구글 시트를 데이터베이스로 활용하여 여행 준비부터 현지 관리까지 올인원으로 관리하는 플래너.
구글 독스에 작성한 여행 고민 로그를 앱에서 불러와 외부 AI(Gemini/Claude)와 상담하는 방식으로 운영.

## 2. 주요 기능

### 2.1. 보안
- **로그인 시스템:** `st.secrets` 기반 비밀번호 인증. 인증 성공 시에만 앱 데이터 로드.

### 2.2. 여행 준비 탭
- **모바일 최적화:** 폰 접속 시 상단 여백 제거(0rem), Streamlit 기본 UI(헤더·하단 footer·`#MainMenu`·`stStatusWidget`·`stToolbar`·뱃지) 전체 CSS 숨김. 제목/버튼 한 줄 배치. 폰트 위계: h1 25px, h2 19px, 본문/버튼 14.5px.
- **예산 관리:** 항공권·숙소·보험 등 항목별 입력. 숙소는 체크인/아웃 날짜로 박수·총금액 자동계산, 무료 취소 기한, 메모.
- **체크리스트:** 준비물 추가/완료 체크, 구글 시트 실시간 반영.
- **예상 일정:** 구글 시트 `상세일정` 탭 연동. ✅ 확정 체크박스 클릭 시 `ok`로 시트 저장. 확정된 항목만 현지 탭 지도에 표시.

### 2.3. 여행 현지 탭
- **1일차~N일차 버튼:** 5일 고정(5/30~6/3). 버튼 전환 시 시트 재조회 없이 빠르게 동작 (session_state 캐시).
- **카테고리 버튼:** ☕ 에어컨카페 / 💆 스파 / 🗺️ 전체동선. memo 확정 데이터 기반.
- **지도:** 확정 일정만 번호 핀(1,2,3...) 표시. 핀 순서대로 점선 연결. 동일 좌표 핀은 오프셋으로 분리 표시. 핀 팝업에 구글맵 링크 포함. 지도 높이는 JS `MutationObserver`로 컨테이너 실제 너비를 감지해 동일하게 설정(정사각형). Python `height=400`은 첫 렌더 fallback용.
- **일정 목록:** 모바일 최적화 3줄 레이아웃 — 번호·시간·내용·소요 / 📍 장소명 버튼(탭 가능) / 메모. 장소명 클릭 시 해당 핀으로 지도 이동 + 화면 스크롤.
- **스크롤:** 장소 클릭 시 "📅 N일차 — 확정 M개" 캡션이 화면 상단으로 이동. scroll 이벤트 리스너 + 900ms 안전망 조합.
- **핀 팝업 연동:** Folium iframe sandbox `allow-popups` 미지원으로 `window.open` SecurityError 영구 불가 확인 → 팝업 내 경비 링크 제거. 핀 팝업에는 구글맵 링크만 유지.
- **지도 아래 경비 버튼:** st_folium `returned_objects`로 핀 클릭 tooltip 감지 → 지도 아래 "💰 장소명 — 경비 기록하기" 버튼 표시. 클릭 시 경비 관리 탭 이동 + 장소명 자동 채움.

### 2.4. 경비 관리 탭
- **내비게이션:** 지도 핀 클릭 → 지도 아래 버튼 → 경비 관리 탭으로 이동 + 장소명 자동 채움 (JS `tabs[2].click()` 방식).
- **입력 방식:**
    *   **직접 입력:** 날짜, 시간, 장소명, 품목, 금액(VND) 입력. 핀에서 넘어온 경우 장소명 자동 채움. 저장 전 품목·금액 유효성 검증(둘 다 빈 값이면 저장 차단). `manual_form_key` 카운터 기반 폼 초기화. 메모 섹션: 헤더 행(**메모** 텍스트 + 🗑️ 버튼) + textarea + 저장 버튼 전체너비.
    *   **영수증 OCR (⚠️ 코드 구현 완료, 동작 미검증):** `utils/ocr.py` — Gemini `gemini-2.5-flash` 연동. 📷 카메라 촬영(`st.camera_input`) / 🖼️ 갤러리 업로드(`st.file_uploader`, jpg/png/webp/heic 지원) 탭 분리로 모바일 대응. 품목별 단가·수량 추출, 합계 검증(영수증 합계 vs 품목 합계 비교), JSON 파싱 완료. 품목명 비한글 시 한글 번역 병기 (예: "Coffee (커피)"). OCR 완료 후 메모 기본값을 `[장소명]OCR`로 설정. API 키는 `st.secrets["GEMINI_API_KEY"]` 에서 읽음.
- **오프라인 큐 모드:** 베트남 느린 인터넷 대응. 상단 🌐 온라인 / ✈️ 오프라인 토글. 저장 시 즉시 로컬 캐시 반영(`queue_update`) + 큐에 적재. 🔄 동기화 버튼 클릭 시 일괄 API 전송(`flush_queue`). 온라인 모드에서는 저장 즉시 플러시.
- **이미지 보관 (❌ 미해결):** `utils/drive.py` 구현 완료이나 서비스 계정의 저장 용량 부족 (`403 Forbidden`) 근본 원인 미해결.
- **최근 경비 내역:** 최근 저장 순 내림차순으로 최대 5그룹 표시. "📊 전체 내역 →" 버튼 클릭 시 JS로 여행 경비내역 탭 전환. 품목 행: 품목명 전체너비 + ✏️수정/🗑️삭제 버튼 50/50 같은 줄.
- **가계부 뷰어:** 영수증ID 기준 그룹핑 표시. 장소·날짜·결제수단·합계 헤더. 품목별 인라인 편집·삭제·추가. 그룹 단위 메모 편집. VND/KRW 환산 표시.

### 2.5. 여행 경비내역 탭
- **일차 필터 버튼:** 1일차(5/30) ~ 5일차(6/3) / 선구매 / 전체. 선택된 버튼 `type="primary"` 강조.
- **선구매 필터:** 여행 일정 날짜(`2026-05-30~06-03`) 이외 날짜 경비만 표시.
- **정렬:** 날짜 + 시간 오름차순 (먼저 지출한 것부터). `pd.to_datetime(날짜+시간)` 파싱으로 시간 문자열 비교 오류 방지.
- **합계 표시:** 필터 기준 VND 합계 + KRW 환산 배너.
- **그룹 뷰어:** 경비 관리 탭과 동일 — 영수증ID 기준 그룹핑, 품목별 인라인 편집·삭제(✏️수정/🗑️삭제 50/50), 그룹 단위 메모 편집.

### 2.6. AI 여행 비서 탭
- 구글 독스 thinklog 실시간 불러오기 (서비스 계정 읽기 전용).
- 헤딩(h1/h2/h3) CSS 소형화로 가독성 개선.
- Gemini / Claude 외부 링크 버튼 제공.

### 2.7. 보안 정책
- 서비스 계정 방식으로 나만 접근 가능한 비공개 구글 시트 및 드라이브 폴더.
- `key.json` 로컬 사용 후 삭제, 깃 포함 금지.
- `secrets.toml`을 통한 보안 설정 관리 시도 중.

## 3. 데이터 구조 (Google Sheets)
- `budget`: category, item, check_in, check_out, nights, price_per_night, cost, cancel_deadline, memo, paid
- `checklist`: item, done
- `hotels`: 이름, 가격, 취소기한, 링크, 장단점
- `상세일정`: 날짜, 요일, 시간, 확정(ok/빈값), 내용, 메모, 장소명, 소요시간, 이동시간, 구글지도, lat, lon, open, close
- `expenses`: 영수증ID, 날짜, 시간, 장소명, 품목, 단가, 수량, 총액(VND), 환산금액(KRW), 결제수단, memo, 저장시간

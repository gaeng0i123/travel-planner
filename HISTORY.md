## 2. 해결된 주요 이슈

| 이슈 | 원인 | 해결 |
|---|---|---|
| 맥 환경 설정 | 경로 문제 | venv + start.sh 쉘 스크립트로 해결 |
| 보안 이슈 | 링크 공유 방식 | 서비스 계정 전용 이메일 편집자 권한으로 전환 |
| Gemini API 할당량 | 결제 활성화 시 무료 할당량 0 | 외부 AI 링크로 대체 |
| TOML 파싱 오류 | 최상단 키가 `[connections.gsheets]` 하위로 인식 | 섹션 위로 이동 |
| 일차 버튼 전체 초기화 | 버튼 클릭 시 매번 시트 재조회 | session_state 캐시로 해결 |
| NaN 표시 | pandas 빈 셀 NaN | `pd.isna()` 체크로 해결 |
| 메모 줄바꿈 미표시 | `st.dataframe` 미지원 | HTML 테이블 + `<br>` 변환 |
| thinklog 헤딩 과대 | Streamlit 기본 h1/h2 크기 | `.thinklog` CSS 클래스로 소형화 |
| 모바일 폰트 엉망 | `span`, `p` 전역 선택자가 Streamlit 내부 요소까지 덮어씀 | `.stMarkdown p, label`로 범위 축소, thinklog CSS에 `!important` 추가 |
| 체크리스트 TypeError | Google Sheets 빈 셀 → NaN → float64, boolean 할당 불가 | done 컬럼 읽을 때 bool로 변환 |
| 모바일 장소 링크 불가 | 6컬럼 표가 모바일에서 터치 불가 | 3줄 레이아웃 + 풀너비 버튼으로 전환 |
| 장소 클릭 스크롤 불안정 | Streamlit 스크롤 복원이 JS 타임아웃보다 늦게 실행 | scroll 이벤트 리스너로 복원 감지 후 즉시 보정 |

## 3. 파일 목록 및 역할
- `app.py`: 메인 웹 애플리케이션 (예산, 체크리스트, 예상일정, 현지 지도, thinklog 뷰어).
- `start.sh`: 맥 로컬 실행 스크립트 (venv 자동 생성 포함).
- `fix_secrets.py`: 구글 서비스 계정 JSON 키 → secrets.toml 자동 변환 도구.
- `.streamlit/secrets.toml`: 구글 시트 서비스 계정 키, 독스 ID, 비밀번호 보관소 (비공개, 깃 제외).

## 4. 브랜치 전략 및 작업 방식
- **기본 원칙:** 모든 신규 기능 개발 및 버그 수정은 `dev` 브랜치에서 진행.
- **배포 프로세스:** `dev`에서 충분히 검증된 작업물만 `main` 브랜치로 머지하여 최종 배포.
- **커밋 관리:** 의미 있는 단위로 커밋하며, `main`은 항상 실행 가능한 최신 배포 상태 유지.

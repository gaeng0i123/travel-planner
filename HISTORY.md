## 2. 해결된 주요 이슈

| 이슈 | 원인 | 해결 |
|---|---|---|
| 맥 환경 설정 | 경로 문제 | venv + start.sh 쉘 스크립트로 해결 |
| 보안 이슈 | 링크 공유 방식 | 서비스 계정 전용 이메일 편집자 권한으로 전환 |
| Gemini API 할당량 | 결제 활성화 시 무료 할당량 0 | 외부 AI 링크로 대체 |
| TOML 파싱 오류 | 최상단 키가 `[connections.gsheets]` 하위로 인식 | 섹션 위로 이동 |
| 일차 버튼 전체 초기화 | 버튼 클릭 시 매번 시트 재조회 | session_state 캐시로 해결 |
| NaN 표시 | pandas 빈 셀 NaN | `pd.isna()` 체크로 해결 |
| 메모 줄바꿈 미표시 | `st.dataframe` 미지원 | HTML 테이블 + `<br>` 변환 |
| thinklog 헤딩 과대 | Streamlit 기본 h1/h2 크기 | `.thinklog` CSS 클래스로 소형화 |
| 모바일 폰트 엉망 | `span`, `p` 전역 선택자가 Streamlit 내부 요소까지 덮어씀 | `.stMarkdown p, label`로 범위 축소, CSS `!important` 추가 |
| 체크리스트 TypeError | Google Sheets 빈 셀 → NaN → float64, boolean 할당 불가 | done 컬럼 읽을 때 bool로 변환 |
| 모바일 장소 링크 불가 | 6컬럼 표가 모바일에서 터치 불가 | 3줄 레이아웃 + 풀너비 버튼으로 전환 |
| 장소 클릭 스크롤 불안정 | Streamlit 스크롤 복원이 JS 타임아웃보다 늦게 실행 | scroll 이벤트 리스너로 복원 감지 후 즉시 보정 |
| 경비 탭 전환 동기화 | URL 파라미터 방식에서 상단·하단 UI 따로 그려짐 | 새 탭 열기 방식으로 우회 (window.close() 복귀) |
| 경비 합계 AttributeError | 시트 컬럼명과 코드 키값 불일치 | 컬럼 존재 여부 확인 후 합계 계산 |

## 5. 파일 목록 및 역할

- `app.py`: 메인 웹 애플리케이션 진입점. URL 파라미터(`?tab=expenses`) 분기 처리.
- `tabs/prep.py`: 여행 준비 탭 (예산, 체크리스트, 예상일정).
- `tabs/trip.py`: 여행 현지 탭 (지도, 일정 목록, 카테고리 버튼).
- `tabs/expenses.py`: 경비 관리 탭 (직접입력, OCR, 가계부 뷰어).
- `tabs/ai.py`: AI 여행 비서 탭 (thinklog 뷰어, 외부 AI 링크).
- `utils/sheets.py`: 구글 시트 연결·읽기·쓰기. `VND_TO_KRW` 상수.
- `utils/auth.py`: 로그인 로직.
- `utils/ocr.py`: Gemini API 기반 영수증 OCR.
- `utils/drive.py`: 구글 드라이브 파일 업로드 (서비스 계정 방식).
- `scripts/fill_coords.py`: 좌표·장소명 자동 추출 스크립트.
- `scripts/fix_secrets.py`: 구글 서비스 계정 JSON 키 → secrets.toml 자동 변환 도구.
- `start.sh`: 맥 로컬 실행 스크립트 (venv 자동 생성 포함).
- `fill.sh`: 좌표·장소명 자동 채우기 단축 실행 스크립트.
- `.streamlit/secrets.toml`: 구글 시트 서비스 계정 키, 독스 ID, 비밀번호 보관소 (비공개, 깃 제외).
